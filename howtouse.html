INSTALLATION

  • Open your favorite code editor.
  • Open folder and select "data_template" *the place where you open this file.
  • Open file 'index.html'.
  • Edit images, text, and link according to your wishes. *please do not change any classes if not necessary
  • When you add item on portofolio please add class 'web', 'photos', or 'design' to categorize them.
  • If you want add new item on portofolio you should also add it on 'img-details-cont' with the same order.
  • You have to register your email address on fromspree to use form. *link on "credit.html" very easy and quick.
  • Save file.

CUSTOMIZATION

  • You're free to change background image/color of main element.
  • You're free to change any color, size of class container and its contents. *changing size impact on responsiveness. so please take carefully
  • You're free to change any font or icon that you have *there is a follow-up to the font you change on index.css

USE

  • To clarify the information, this product is not available for internet explorer users due to several reasons. So make sure the browser you are using is compatible.
  • You cannot change any element other than those mentioned in customization.
